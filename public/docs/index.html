Customer Register

Example request:
curl --request POST \
    "https://medi-order.herokuapp.com/api/auth/customer/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"eac\",
    \"email\": \"pbayer@example.org\",
    \"password\": \"\",
    \"phone\": \"mwiutfikjznpl\",
    \"gender\": \"male\",
    \"location\": \"enim\"
}"
